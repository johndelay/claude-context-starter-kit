# projects/

This folder holds your active work. One subfolder per project.

Each project folder contains:
- CLAUDE.md              — what the project is, decisions made, current status
- CHANGELOG.md           — project-specific history
- SESSION_YYYY-MM-DD.md  — session records (most recent = your recovery point)
- documents/             — docs, drafts, and research Claude produced

## Starting a new project
Create a subfolder with the project name. Copy the structure above.
Claude can do this for you — just say "start a new project."

## When a project is done
Move the folder to z-archive/. Don't delete it.
