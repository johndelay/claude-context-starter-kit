# scratch/

For quick, one-off tasks that don't need a project folder.

Use this for:
- One-time emails or drafts
- Quick research that won't be revisited
- Anything where you just need a file and won't be resuming

No SESSION files needed here. Work in scratch/ doesn't compound —
that's by design. If something in scratch/ turns into real work,
move it to projects/ and give it a proper folder.
